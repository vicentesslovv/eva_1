# Blupy Flowers - Servicios de Jardinería 

## Funcionalidades implementadas 
* **Index:** Estructura semántica con Flexbox y banner Full Width.
* **Galería:** Catálogo de productos organizado en cuadrícula de 3x2 con descripciones.
* **Login:** Formulario profesional con diseño gris oscuro y validación de campos.
* **Diseño Responsivo:** Adaptabilidad para diferentes tamaños de pantalla.

##  Tecnologías utilizadas
* HTML5 semántico.
* CSS3 avanzado (Flexbox, Gradients, Box-shadow).
* Git/GitHub para el control de versiones.
